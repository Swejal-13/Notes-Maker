# 📝 Notes Maker App (React)

A simple and responsive **Notes Making Application** built using **React.js**.  
This app allows users to **add, view, and delete notes** easily with a clean UI.

---

## 🚀 Features

- ➕ Add new notes
- 🗑️ Delete notes
- 📋 View all saved notes
- ⚡ Fast and responsive UI
- 🔄 Real-time updates without page reload

---

## 🛠️ Technologies Used

- **React.js** – Frontend framework
- **JavaScript (ES6+)**
- **HTML5**
- **CSS3**

---
## 🎯 How It Works

1. Enter your note in the input field
2. Click on **Add** button
3. Note will be displayed instantly
4. Click **Delete** to remove a note
